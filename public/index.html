Easy-typescript-documentation

Documentation for the package easy-typescript-documentation.

@Doc

Register a class to be used by the functions.
Nothing will be outputed by the functions if this is not present.

@ClassName

Take a string as argument.
Add a name to a class to be displayed.

@ClassDesc

Take a string as argument.
Add a description to a class to be displayed.

@Name

Take a string as argument.
Add a name to a propertie to be displayed.

@Desc

Take a string as argument.
Add a description to a propertie to be displayed.

@Note

Take a string as argument.
Add a note to a propertie to be displayed.

@Color

Take a DOC_COLOR as argument. ('red','orange','grey','none')
Add a color highlight to a propertie.

getJson

Take an Object as argument.
Output a list of object representing the objects decorated by @Doc in the argument object.
The structure is available here.

getDocPage

Take a name (string), an object containing class decorated with @Dodc and an optonal description (string) as arguments.
Output an html page as a string.

createClassAnnotation

Take up to 3 string argument.
Those arguements can be typed like so: createClassAnnotation("arg0","arg1","arg3").
Output a decorator for a class that will take as many argument as specified.
The string arguments will be the key at which the values will be available in the json generated by getJson.

createPropertieAnnotation

Take up to 3 string argument.
Those arguements can be typed like so: createPropertieAnnotation("arg0","arg1","arg3").
Output a decorator for a arguments that will take as many argument as specified.
The string arguments will be the key at which the values will be available in the json generated by getJson.
